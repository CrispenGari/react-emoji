### React Emojify
This is a React Component Library for rendering emoji components.

### Getting started
To use `react-emojify` you need to install it as follows:

1. using `npm`
```shell
npm i @crispengari/react-emojify
```

2. using `yarn`
```shell
yarn add @crispengari/react-emojify
```
